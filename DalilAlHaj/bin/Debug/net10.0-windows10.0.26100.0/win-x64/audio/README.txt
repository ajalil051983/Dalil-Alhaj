Audio Files Placeholder
========================

This folder should contain MP3 audio files for each subcategory.

File Naming Convention:
- Format: {categoryId}_{subcategoryId}.mp3
- Example: 1_101.mp3 (Category 1, Subcategory 101)

To add audio files:
1. Record or obtain Arabic audio narrations for each subcategory content
2. Convert to MP3 format
3. Name according to the convention above
4. Place in this folder

Current Status:
- All 24 subcategories are marked with hasAudio: true
- Audio files are not yet available
- App will show audio player but files need to be added

Audio Requirements:
- Format: MP3
- Quality: 128-256 kbps recommended
- Language: Arabic narration matching the content
- Clear pronunciation following Maliki school terminology
